Multi-User Inventory Tracker

Need to track inventory with multiple people making adjustments at once? Stop relying on sticky notes, shared spreadsheets that overwrite each other, or manual counts that are already wrong by the time you finish. This tracker logs every move — who did it, when, and where it went — automatically.

What This File Does

  • Tracks inventory items by ID, description, quantity, and physical bin location
  • Supports ADD, SUBTRACT, and BIN-TO-BIN movements from a single entry form
  • Automatically stamps every transaction with the date, time, and user name
  • Maintains a permanent History Log — nothing gets deleted or overwritten
  • Dropdown-driven data entry reduces typos and keeps records consistent
  • Password protected to prevent accidental changes to structure or formulas

How to Use It

  1. Open in Excel desktop app and click Enable Macros when prompted
  2. Go to the Transactions tab and fill in the movement fields
  3. Select a Movement Type from the dropdown (Add, Subtract, Bin to Bin)
  4. Click the action button — the macro processes and records the entry
  5. Check History Log to see the timestamped record
  6. View Main Inventory to see updated quantities in real time
🧪 The Challenge Try to break it. Enter bad data, skip fields, click buttons out of order. It's built to handle real-world use — including the mistakes. See how it responds, and imagine what that reliability could mean for your team.

Excel · VBA · Bill of Materials

Bill of Materials Filter

Ever needed to know which finished products contain a specific component — or pull every part in a kit instantly? This tool works both ways. Type a part number and see every product it belongs to. Type a product and see every component inside it. Search multiple items at once using a simple comma-separated entry.

What This File Does

  • Instantly filters a Bill of Materials by Part Number, Set ID, or Project Name
  • Works bidirectionally — search by component to find kits, or by kit to find components
  • Supports partial matches — no need to type exact IDs
  • Multi-item search using comma separation (e.g. "PLR-0040,WRN-0030")
  • Displays quantity per set, In Set Flag, Critical Flag, and list price per result
  • BoM data lives on a separate sheet — easy to update without touching the filter logic

How to Use It

  1. Open in Excel desktop app and click Enable Macros when prompted
  2. Go to the BoM Filter tab
  3. Type a value into a filter cell — use a real Part Number or Set ID from the BoM tab as your starting point
  4. Click the Search button — results will populate based on your entry
  5. To search multiple items at once, separate values with a comma and no space (e.g. PLR-0040,WRN-0030)
  6. Clear all filter fields to reset and start a new search
🧪 Try This Type SFY-0141 in the Part Number field — it appears in 14 different kits. Then clear it and type KIT-500 in the Set ID field to see every one of its 32 components. Finally, try searching PLR-0040,WRN-0034 at once to find every kit that contains both parts.
